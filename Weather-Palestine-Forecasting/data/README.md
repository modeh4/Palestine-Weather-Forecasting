Dataset: final_filled_climate.xlsx

This file contains the consolidated daily meteorological observations used in this study.
It includes data collected from five weather stations in Palestine between 2015 and 2025.

Variables:
- Date
- Station
- Temperature (°C)
- Relative Humidity (%)
- Wind Speed (m/s)
- Rainfall (mm)
- Atmospheric Pressure (hPa)
- Sunshine Duration (hours)

This dataset serves as the input for preprocessing, feature engineering, and all forecasting models evaluated in the study.

Dataset: final_filled_climate_cleaned.xlsx
This file contains the  daily meteorological observations used in this study after preprocessing.

Datasets: TEMPERATURE.xlsx, RAIN.xlxs, WIND SPEED.xlsx, SUNSHINE.xlsx, PRESSURE.xlsx, RELATIVE HUMIDITY.xlsx. 
These files contain raw data from Palestinian Meteorological Authority.
